border-radius属性的值的顺序 border-top-left-radius;border-top-right-radius;border-bottom-right-radius;border-bottom-left-radius;

1.画一个半圆

.box{
width:200px;
height:100px;
background-color:red;
border-radius:200px 200px 0 0;
}

2.画一个四分之一圆

.box{
width:200px;
height:200px;
background-color:red;
border-radius:200px 0 0 0;
}

3.水平半径与垂直半径

border-radius给我们提供了这样的方案：
border-radius：水平半径 ／ 垂直半径;

4.画一个椭圆

.box{
width:200px;
height:100px;
background-color:red;
border-radius:100px / 50px;
}

5.画一个半椭圆

.box{
width:200px;
height:100px;
background-color:red;
border-radius:200px 0 0 200px / 50px 0 0 50px;
}

6.画一个树叶
